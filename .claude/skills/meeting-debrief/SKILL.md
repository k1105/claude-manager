# 打ち合わせデブリーフ

打ち合わせ終了後に実行する。ユーザーに決定事項・TODOを確認し、記録に残す。

## 引数

- 打ち合わせ名（必須）: スキル呼び出し時に渡される

## フロー

1. `config/channels.json` からdaily-briefingチャンネルIDを取得し、ユーザーに聞く:
   - 「打ち合わせ「{打ち合わせ名}」お疲れさまでした。決まったこと・新しいTODOがあれば教えてください」
2. 返答を待つ（1回聞いて終わり。返答がなければ追わない）
3. 返答があった場合:
   - **TODOがあれば** → `tasks/YYYY-MM-DD.md` に追記
   - **決定事項があれば** → `logs/daily/YYYY-MM-DD.md` に記録
   - **関連するプロジェクトファイルを更新する**
4. 記録完了を伝える

## 注意

- 1回聞いて返答がなければ終了
- 深掘りしない。事実の記録に徹する
