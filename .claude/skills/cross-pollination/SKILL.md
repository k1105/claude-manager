# 異業種クロスポリネーション

プロジェクトやアイデア発想に使える、異業種の面白い事例を収集する。同業種・近い領域は絶対に調べない。

## 手順

1. 以下のジャンルリストからランダムに1つ選ぶ（毎回違うものを選ぶ）：
   - 生物学・進化論 / 都市計画・建築 / 料理・食文化 / スポーツ科学 / 宗教・儀式 / ボードゲーム・カードゲーム / ファッション / 考古学・人類学 / 音楽理論 / 数学・幾何学 / 映画・演劇 / 哲学 / 経済学・行動経済学 / 医療・看護 / 農業・園芸 / 軍事戦略 / 言語学 / 天文学 / 手品・イリュージョン / ダンス・舞踏 / 玩具・遊具 / 交通・物流 / 気象学 / 香水・調香 / 図書館学 / 祭り・カーニバル / 動物の行動学 / 素材科学 / 漫才・落語 / 地図学

2. そのジャンルで「面白い」事例を1つWebSearchで探す
   - 「なぜそれが成立しているのか」まで掘る
   - 出典（URL・著者・年）を必ず付ける

3. 見つけた事例を `research/cross-pollination.md` に追記する（日付付き）

4. **Discordには投げない**。ニュース共有のタイミングでまとめて共有する

## 原則

- **同業種・近い領域は絶対に調べない**
- **掛け合わせはユーザーに委ねる**。面白い事例を置くだけ
- **量より質**。1つの事例を深く掘る
