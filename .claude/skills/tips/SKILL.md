# Tips

`tips/` ディレクトリに格納されたガイドを一覧表示し、ユーザーが選んだtipsの手順に沿ってセットアップや設定を実行する。

## 引数なし → 一覧表示

`tips/` ディレクトリ内の全 `.md` ファイルを読み、各ファイルのタイトル（`# ` 行）と概要を一覧で表示する。

表示例：
```
📚 利用可能なTips：

1. **GCPの仮想マシンで常時稼働させる** — Claude Codeセッションをクラウド上で24時間稼働
2. **Obsidianでドキュメントを可視化する** — Markdownファイルをグラフビューで俯瞰
3. **PC起動時にセッションを自動起動する** — 再起動後も自動でDiscord対話を復旧

実行したいものがあれば番号かタイトルで教えてください。
```

## 引数あり → 実行

引数（番号、ファイル名、またはキーワード）に該当するtipsファイルを読み込み、手順に沿って実行・案内する。

- 手順に実行可能なコマンドがあれば、ユーザーに確認の上で実行する
- 手順が手動操作を必要とする場合は、ステップごとにガイドする
- 完了したら結果を報告する

## 注意

- tipsファイルは `tips/` ディレクトリ内の `.md` ファイルのみ対象
- 新しいtipsを追加するときは `tips/` に `.md` ファイルを作成するだけでよい
